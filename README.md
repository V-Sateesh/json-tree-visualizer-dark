
# json-tree-visualizer-dark


JSON Tree Visualizer
A simple and interactive web app to visualize JSON data as a tree. Built using React, Vite, React
Flow, and Tailwind CSS.


Live Demo 
https://json-tree-visualizer-dark.vercel.app/


Features
• Paste JSON and visualize instantly

• Detects invalid JSON and shows errors

• Color-coded nodes (Objects, Arrays, Primitives)

• Search JSON path (e.g., $.user.name)

• Highlight matched node

• Click node to copy JSON path

• Download visualization as PNG

• Dark  / Light mode toggle


How to Run Locally
• 1. Clone the repo: git clone https://github.com/V-Sateesh/json-tree-visualizer-dark.git

• 2. Navigate to the folder: cd json-tree-visualizer-dark

• 3. Install dependencies: npm install

• 4. Start the server: npm run dev

• Then open http://localhost:5173/ in your browser.


Built With
• React + Vite — for fast frontend development

• Tailwind CSS — for modern styling

• React Flow — for visual node-based graphs

• HTML-to-Image — to download as PNG
